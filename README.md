##  Featured Projects

### 1. Application Security Assessment: Cross-Site Scripting (XSS)
**Focus**: Web Application Security | OWASP Top 10
**Summary**:  
Conducted a controlled XSS assessment to identify input validation vulnerabilities in a web application. Tested for reflected and stored XSS, analyzed impact on session security, and provided remediation aligned to OWASP and secure coding standards.

**Key Skills**: `OWASP` `Burp Suite` `Vulnerability Testing` `Secure Coding` `Risk Reporting`
**Artifacts**: `XSS_PoC` `Risk Ratings` `Remediation Plan`
